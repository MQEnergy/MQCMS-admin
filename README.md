# MQCMS-admin
MQCMS是一款现代化，快速，高效，灵活，前后端分离，扩展性强的CMS系统。
MQCMS中的MQ取麻雀拼音首字母。寓意麻雀虽小五脏俱全，本项目使用iview vuejs组件搭建的后台系统模块。

后端项目：https://github.com/MQEnergy/MQCMS
